# 交通-电力耦合网络灾后抢修调度与韧性提升 - 实验总结报告

## 1. 实验背景与动机

极端天气与地质灾害（如滑坡、泥石流等）往往同时造成道路阻断与配电线路故障，引发交通网络与电力网络的相依失效。灾后恢复的核心难点在于“交通对电的物理可达性硬约束”以及“多应急资源的协同调度”。

本实验基于预先设定的交通与电力双层耦合网络拓扑，将灾后抢修调度抽象为一个事件驱动模型。模型考虑了“道路抢修队伍、电力抢修队伍、直升机空中支援”等异构资源，并在统一的框架下执行了4种基线策略，通过监控关键设施供电率恢复曲线、累计收益（AUC）及工期等指标，对比各类策略的韧性提升效果。

## 2. 调度策略体系设计

针对多智能体协同，本实验设计了四类调度策略，主要差异体现在“恢复早晚”对早期韧性收益的影响上：

- **策略 S1（先路后电）：** 优先抢通关键通道，待道路可达后再由电力队伍进入修复。可解释性强，但恢复周期较长。
- **策略 S2（边路边电，并行协同）：** 修路与修电队伍并行推进。对于无法乘车到达的故障点，采用“徒步携带”模式强行修复，承受较高的效率衰减。
- **策略 S3（空中响应）：** 忽略地面道路瓶颈，主要依靠直升机资源跨越障碍直达孤岛故障点。
- **策略 S4（天地协同）：** 结合地面修路与直升机投送机制。直升机集中支援高价值、短期无法由地面快速到达的关键负荷，将可达性瓶颈转化为灵活的绕行支持。

## 3. 实验结果与可视化对比

本次实验通过基于 Python 和 NetworkX 构建的仿真器复现，并将指标同步日志至 Weights & Biases (WandB)。随后提取相关数据进行可视化处理。核心评估口径为：
1. **LSD（已恢复负荷曲线，kW）**
2. **AUC（累计供电收益，kW·h）**
3. **Makespan（工期，h）**

### 3.1 供电率恢复曲线 (LSD)
阶梯曲线直观反映了不同抢修策略在每个时刻的已恢复负荷容量。

![LSD 恢复曲线](https://i.ibb.co/MknT80z8/6b0ab6be1f99.png)

*观察分析：* S1 存在较长的停电平台期，前期进度缓慢。S3 与 S4 曲线能够迅速拉升并在前期抢通大量关键负荷，体现了空中资源的快速干预价值。

### 3.2 累计供电收益对比 (AUC)
为了综合衡量“越早恢复越好”的韧性标准，采用曲线积分面积（AUC）进行量化。

![AUC 柱状图](https://i.ibb.co/rGC85npx/dae171b66305.png)

*观察分析：* S4 (天地协同) 的累计收益最高，突破了单网串行恢复的局限，比传统保守的 S1 策略（先路后电）提升了近一倍的韧性收益。

### 3.3 工期对比 (Makespan)
衡量完成所有灾后网络恢复任务的总耗时。

![工期柱状图](https://i.ibb.co/0VVKpg82/38249d0c7f8b.png)

*观察分析：* 尽管 S2 的总工期可能是最短的（多线并行推进迅速），但其前期关键负荷的恢复速度不如带有空中干预的 S4。S1由于采用了串行的等候逻辑，导致工期最长。

## 4. 结论与总结

1. **早期恢复优先性：** 在最终完全恢复的总容量一致的情况下，评价指标需要高度侧重于“早期关键点”的恢复收益（AUC）。
2. **直升机的边际贡献：** 直升机的存在将原本绝对的道路硬阻断转换成了可规避的“软阻碍”，极大避免了长时间的被动等待，对抢占“救援黄金时间”有决定性意义。
3. **天地协同框架（S4）的最佳表现：** 地面队伍的铺开保证了覆盖率，而直升机的介入则加速了关键瓶颈的突破。S4 在控制合理总工期的同时，实现了最大的累计韧性收益，是未来大型灾后复杂场景中的推荐基线指导模型。
